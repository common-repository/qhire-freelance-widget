This plugin adds a widget which displays freelance status, skills, description, portfolio link and a contact form.

== Description ==
This plugin provides a great way to monetize design and development blogs. This plugin looks great and attracts potential employers.

###Donation

You can donate to this plugin by purchasing a course at [QScutter](http://qscutter.com)

== Installation ==
1. Upload the qhire folder to the /wp-content/plugins/ directory.

2. Activate the plugin via the \'Plugins\' menu in WordPress.

3. Plugin settings are attached to the widget admin panel form.
